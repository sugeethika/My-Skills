Tech Glossary

Python
Python is a high-level, interpreted, and general-purpose programming language known for its emphasis on code readability and simplicity. Created by Guido van Rossum and first released in 1991, it has become one of the most popular languages in the world due to its versatility and beginner-friendly nature.
SQL
SQL (Structured Query Language) is the standard programming language used to define, manage, and manipulate data stored in relational databases. It is widely used by developers, data analysts, and database administrators to interact with data in a structured, table-based format.
My SQL
MySQL is an open-source relational database management system (RDBMS) that uses Structured Query Language (SQL) to store, manage, and retrieve data. It is widely used by major companies like Facebook, Netflix, and Uber due to its reliability, scalability, and high performance.
HTML
HyperText Markup Language (HTML) is the standard markup language used to create and structure content on the World Wide Web. It is a foundational technology for web development, working alongside Cascading Style Sheets (CSS) for visual presentation and JavaScript for interactivity.
